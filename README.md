## **About Me**

I’m an AI Research Engineer at KakaoBank, building reasoning-focused LLM systems for production services used by 26M+ users.

My core work is post-training and inference optimization for 200B+ models, with a focus on balancing quality, latency, and reliability.

I design scalable reasoning agents that combine tool use, function calling, and structured memory for multi-step tasks in financial and service domains.

Current interests:
- Parallel and continuous reasoning beyond token-by-token generation
- Episodic and structured memory for long-horizon context
- Alignment and evaluation for consistency, calibration, and transparent justification

I’m open to collaboration on reasoning, memory, and trustworthy AI.
Contact: yhlee.nlp [at] gmail.com
